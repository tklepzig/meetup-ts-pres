Some Best Practices

Always define type when initialising an object with some properties

interface Person {
  id: string;
  name: string;
}
const person = { name: "Igor" }; // everything seems fine
const person: Person = { 
  name: "Igor"
}; // missing property 'id'

When using connect HOC, specify all generic parameters explicitly:

connect<Props, DispatchProps, OwnProps, GlobalState>(
mapStateToProps,
mapDispatchToProps,
)(ScheduleBody);
